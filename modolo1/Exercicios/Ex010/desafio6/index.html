Desafio das Tags

Agora é a hora de mostrar o que você está aprendendo de HTML5. Esse exercício faz uma lista de coisas que podem ser adicionadas em uma página. O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag ultilizada para fazer isso;

  • Paragrafos
  • Títulos
  • Imagens
  • Abreviações
  • Codigo-fonte
  • Listas numeradas
  • Listas com desmacadores
